# C2: AGGREGATOR

**C2: AGGREGATOR** — это пользовательский скрипт для платформы Cosplay2, который собирает все мероприятия и фестивали в единый интерфейс. Позволяет фильтровать события по городам, рассчитывать расстояние для поездок и отслеживать дедлайны подачи заявок.
